# College Mini Projects - Python Dependencies
# 
# This project uses only Python standard library modules.
# No external dependencies are required.
# 
# Python Version: 3.6+
# 
# Standard library modules used:
# - sys (system-specific parameters and functions)
# - os (operating system interface)
# - json (JSON encoder and decoder)
# - datetime (date and time handling)
# 
# To run these projects, ensure you have Python 3.6 or higher installed:
# python --version
# 
# All projects can be run directly without installing additional packages:
# python calculator/calculator.py
# python student_grade_system/grade_system.py
